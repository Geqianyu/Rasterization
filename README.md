# Rasterization
## 配置
- 操作系统: win 10
- IDE: vs2022
- 第三方库: stb_image.h, 用于加载纹理
- 界面显示: 利用 WINAPI
- 多线程: 利用 omp
- 模型: 位于文件夹 obj 中

## 说明
- 本项目完成一个软光栅化器，用C++完成
- 按住鼠标左键可以旋转模型，按住鼠标右键可以拖拽模型
- 项目采用 blinphong 光照模型
